if ... elif ... else

In this article, we will learn about the Python if ... elif ... else statement, which is used to create decision-making programs based on different conditions. We will also see some code examples to illustrate how this statement works.

The Python if ... elif ... else statement is a way of writing multiple if statements in a concise and readable way. The syntax of this statement is:


The if ... elif ... else statement evaluates each condition from top to bottom until it finds one that is True, and then executes the corresponding block of code. If none of the conditions are True, it executes the else block (if present).

Let's see some examples of using the Python if ... elif ... else statement.

We will write a program that asks the user to enter a number and prints if it is positive, negative or zero.

Example 1: 

  # ask the user to enter a number
  number = int(input("Enter a number: "))

  # check if the number is positive, negative or zero
  if number > 0:
      print("The number is positive.")
  elif number < 0:
      print("The number is negative.")
  else:
      print("The number is zero.")

                  
Output:

  Enter a number: 5
    The number is positive.

  Enter a number: -3
    The number is negative.

  Enter a number: 0
    The number is zero.


Let's try one more example.

We will write a program that asks the user to enter their grade and prints their performance based on the following criteria:

  • A grade above 90 is excellent
  • A grade between 80 and 90 is good
  • A grade between 70 and 80 is average
  • A grade below 70 is poor

Example 2: 
  
  # ask the user to enter their grade
  grade = int(input("Enter your grade: "))
  
  # check the performance based on the grade
  if grade > 90:
      print("Your performance is excellent.")
  elif grade >= 80 and grade <= 90:
      print("Your performance is good.")
  elif grade >= 70 and grade < 80:
      print("Your performance is average.")
  else:
      print("Your performance is poor.")

                  
Output:

  Enter your grade: 95
    Your performance is excellent.
                  
  Enter your grade: 85
    Your performance is good.
                  
  Enter your grade: 75
    Your performance is average.
                  
  Enter your grade: 65
    Your performance is poor.


The if ... elif ... else statement is a powerful tool in Python that allows you to make decisions based on conditions. By using these statements, you can write code that is flexible and can adapt to different situations. It is a useful tool for creating decision-making programs in Python. Whether you're building a web application, analyzing data, or working on a scientific computing project, the if ... elif ... else statement is an essential feature of Python that you should master.
