Python If ... Else statement

  • Odin
  • 2024 July 25
The `if` and `else` are control flow statements

It allow Python to make decisions based on conditions.

In this following example, the expression `z > 5` checks if the value stored in variable `z` is greater than 5.

If it's true, Python executes the code inside the indented block under 'if'.

Otherwise, it runs the code under 'else'.


z = 10
if z > 5:
  print('z is greater than 5.')
else:
  print('z is not greater than 5.')
Let's break down this piece of code:

z = 10 # Here we're declaring a variable 'z' and assigning it the value 10.

if z > 5: # We're setting up an if-else statement.

# The 'if' part checks whether the condition (in this case, 'z is greater than 5') is true.

print('z is greater than 5.') # If the condition in parentheses is true, Python executes this line of code.

else: # If the condition in the if statement was false, Python will execute what's under else instead.

print('z is not greater than 5.') # This line gets executed only if the 'if' condition was false (i.e., z wasn't greater than 5).

The text inside the parentheses ('z is greater than 5.') is what gets printed:

z is greater than 5.

This is a simple way for Python to make decisions based on conditions.

In conclusion

This code first declares a variable `z` and assigns it the value of 10.

Then, it uses an `if-else` statement to check whether `z` is greater than 5.

If that's true, it prints 'z is greater than 5.'; if not, it prints 'z is not greater than 5.'.
